# RestoAM Workorders Frontend

React + Vite frontend for managing RestoAM Workorders.

## Configure API

```
VITE_WORKORDER_API=http://localhost:8082/restoam/workorders
VITE_ASSET_APP_URL=http://localhost:5173
VITE_LOCATION_APP_URL=http://localhost:5174
```

## Run

```bash
npm install
npm run dev
```

## Tests

```bash
npm run test:e2e
```
